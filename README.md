# Student Grade Tracker

A simple Java console-based application to manage student grades.

## Features
- Add student details
- Store marks using ArrayList
- Calculate average marks
- Find highest and lowest scores
- Display student summary report
- Exception handling for invalid input

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Exception Handling

## Project Structure
- Main.java
- Student.java
- GradeManager.java

## How to Run
1. Open project in IntelliJ IDEA
2. Run Main.java
3. Use menu options in console
